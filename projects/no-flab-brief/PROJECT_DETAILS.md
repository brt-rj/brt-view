# No-Flab Brief

**Automated News Aggregation & Verification System**

This project aggregates news from over 30 global sources, filters them using AI for a balanced diet of 40% Politics and 60% Other beats, and verifies facts using a Ruthless Editor AI agent.

- **Stack**: Python, Supabase, Observable Framework, GitHub Actions
- **Data**: RSS, YouTube, GDELT, Hacker News
